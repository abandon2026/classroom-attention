# 课堂专注度分析系统

## 项目简介
本项目针对传统课堂管理难以实时掌握学生听课状态的痛点，设计并实现了一套**基于多任务卷积神经网络的课堂专注度智能分析系统**。

## 核心功能
- 🎯 **人脸检测与识别**：实时检测课堂人脸，识别学生身份
- 🧠 **头部姿态估计**：通过6D-RepNet分析学生抬头/低头/转头
- 😊 **表情识别**：识别专注、困惑、疲倦、分心等表情状态
- 🙋 **行为检测**：检测举手、玩手机、趴桌、写笔记等课堂行为
- 📊 **专注度评分**：综合多模态信息，输出0-100分专注度
- 📈 **数据报表**：自动生成课堂分析报告

## 应用场景
- 高校智慧教室建设
- 在线教育质量评估
- 教师教学效果反馈
- 学生行为分析研究

## 技术栈
| 模块 | 技术选型 |
|------|---------|
| 人脸检测 | MTCNN |
| 人脸识别 | FaceNet |
| 头部姿态 | 6D-RepNet |
| 表情识别 | mini_Xception |
| 行为检测 | YOLOv8 |
| 后端框架 | Flask |
| 前端界面 | PyQt5 |
| 数据库 | SQLite |
